Please enter four different addresses or names to form a 4-point polygon (a path). Objects inside this polygon will be detected for later analysis.













Please specify a path color either as a 24-bit (example: 0xFFFFCC) or 32-bit hexadecimal value (example: 0xFFFFCCFF), or from the set {black, brown, green, purple, yellow, blue, gray, orange, red, white}




Please specify a path thickness, 2 or 3 is recommended.




Maps on Google Maps have an integer 'zoom level' which defines the resolution of the current view. Zoom levels between 0 (the lowest zoom level, in which the entire world can be seen on one map) and 21+ (down to streets and individual buildings) are possible within the default roadmap view. Building outlines, where available, appear on the map around zoom level 17. This value differs from area to area and can change over time as the data evolves.




Maximum allowable values for the size of the map is 640x640 pixels.




Available map types: roadmap, satellite, terrain, hybrid.
